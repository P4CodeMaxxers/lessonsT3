# Test Project Documentation

This project serves as a test case for the `_projects` integration system.

## Features
- **Automated JS Distribution**: Scripts in `js/` are automatically synced to the site's assets.
- **Native SASS Support**: Styles in `sass/` are integrated into the Jekyll asset pipeline.
- **Auto-Registration**: Controlled via `_projects/.makeprojects`.

See the [Project Notebook](../) for a detailed technical breakdown.
